# Phising_Detect

Python Tkinter 기반 피싱 문자 탐지 GUI 프로그램입니다. 문자 내용과 발신번호를 입력하면 URL, 도메인, 위험 키워드, 발신번호, 위험 조합을 분석해 점수를 계산하고 `정상 가능성 높음 / 의심 / 위험 / 고위험`으로 분류합니다.

## 핵심 기능

- 문자 본문과 발신번호 기반 피싱 위험도 분석
- URL 포함 여부, 단축 URL, IP 주소 URL, 무료 호스팅, 의심 TLD 탐지
- 공식 도메인은 정상 확정이 아니라 감점 신호로만 반영
- 브랜드명과 실제 URL 도메인이 다른 경우 위험 점수 부여
- 개인정보, 인증번호, 앱 설치, 원격제어, 긴급성, 금전 유인 키워드 탐지
- 정상 인증번호 안내처럼 오탐이 잦은 문구는 안전 안내 문맥을 함께 확인
- Discord/Steam 등 영문 이벤트 사칭 문자 탐지
- SQLite DB에 키워드, 공식 도메인, 예시 문자, 검사 로그 저장
- Tkinter GUI에서 결과 카드, 점수 바, URL 표, 탐지 근거, 결과 복사 제공

## 실행 방법

```bash
python app.py
```

```bash
python -m unittest -q

```

## 배포 참고

Windows exe로 빌드할 경우 `phishing_rules.db`는 실행 파일과 같은 폴더에 자동 생성됩니다. 검사 로그가 포함될 수 있으므로 DB 파일은 GitHub에 업로드하지 않습니다.

## 등급 기준

- 0~2점: 정상 가능성 높음
- 3~5점: 의심
- 6~8점: 위험
- 9점 이상: 고위험
